Backend-API
API RESTful com Node.js e Express implementando CRUD.

Para acessar o banco de dados no mongodb atlas será preciso criar um arquivo .env com as seguintes informações:

MONGODB_USER=usrTarefas
MONGODB_PASSWORD=abcd1234
MONGODB_HOST=cluster0.ajfevzi.mongodb.net
MONGODB_DATABASE=Voluntario
JWT_SECRET=segredo
JWT_EXPIRES=60
